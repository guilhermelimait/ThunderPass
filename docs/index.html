What's inside

Everything a Thunder Traveler needs

ThunderPass packs a full feature set — from low-power BLE discovery to retro game sync, all running silently in the background.

📡

BLE StreetPass Encounters

Scan and advertise simultaneously using Bluetooth Low Energy. When two ThunderPass users are nearby, a "Spark" happens automatically — no tapping required.

⚡

Joules Energy System

Earn 100 Joules for every unique Spark. Spend Joules in the Visual Shop to unlock "CRT Scanlines", "Pixelated Aura" and "Thunder Trail" profile effects.

🎮

RetroAchievements Sync

Link your RetroAchievements account. After every BLE exchange, see your peer's rank, points and recent game masteries right on their Spark Card.

🏆

Badges & Achievements

Unlock 12 unique badge types — from First Spark to Legendary Encounter. Badges glow grey → blue → purple → orange → gold as you level up.

👻

Ghost Challenges

Share your best score or time from any game as a "Ghost". When you meet another player, compare scores and settle who reigns supreme.

🔒

End-to-End Encrypted

Every BLE exchange is protected with ECDH P-256 key agreement, AES-256-GCM authenticated encryption, and Keystore-bound ECDSA signatures. Fully offline — no accounts, no servers, no data ever leaves your device unencrypted.

Roadmap

What's done & what's next

Five milestones from zero to cloud. Milestones 0–4 are fully shipped. Milestone 5 is the platform frontier.

M0
Repository + Android App Scaffold ✅ Complete
✓Android project scaffold (Kotlin + Jetpack Compose)
✓Foreground service (BleService) + persistent notification
✓Runtime permission flow for Android 13 BLE + notifications
✓Local storage with Room (MyProfile, Encounter, PeerProfileSnapshot)
✓CI workflow — debug APK built on every push/PR
M1
StreetPass MVP ✅ Complete
✓BLE advertise + scan (concurrent, low-power)
✓Rotating ID via HMAC-SHA256 — never broadcasts raw install ID
✓Encounter dedup (10-min cooldown per rotating ID)
✓GATT server + client — profile card exchange over BLE
✓Brand theme (ThunderYellow + ThunderGray, dark/light, adaptive icon)
✓Unit tests: RotatingIdUtils, EncounterDedup
M2
Trust, Quality + Energy Base ✅ Complete
✓Home screen widget — new encounters indicator
✓Joules energy engine — 100 J per unique Spark
✓Haptic feedback — double-pulse vibration on successful exchange
✓Battery modes: aggressive, balanced, off
✓Better UX copy + onboarding (The "Grid" introduction)
M3
Fun Layer + Game Sync ✅ Complete
✓Ghost payload type (share your best score/time via BLE)
✓RetroAchievements integration — fetch peer rank, points, masteries
✓RetroSparkCard — show RA rank + points on encounter cards
✓Mastery icons row on encounter detail view
✓Platinum Pulse, Legendary Encounter, Retro Circuit badges
✓Visual Shop — unlock effects with Joules (CRT, Pixelated Aura, Thunder Trail)
M4
Security + Device Sync ✅ Complete
✓End-to-end AES-256-GCM encryption (ECDH P-256 + HKDF key agreement, fresh keys per session)
✓ECDSA P-256 payload signatures — Keystore-bound, private key never leaves the device
✓Privacy Mode — omits stable identity fields from BLE payload on demand
✓Two-device SAS pairing (Short Authentication String over BLE)
✓Device Group Key — 15-min ambient delta-merge sync between paired devices
✓Walking Volts — hardware step counter converts steps to energy (100 steps = 1 Volt)
✓Home screen widgets — 2×1 BLE toggle + 2×2 stats card (Jetpack Glance)
M5
Platform — The Future 🔭 Planned
○Optional cloud backup / restore (encounter history, opt-in)
○Public "payload provider" API / SDK for third-party apps
○Power Surge Events — location-based 2× Energy multipliers
○Versioned protocol extensions
○OTA update delivery from the app
